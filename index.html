1.Tanpa Generic

  • Pada program ini, Box merupakan class yang digunakan untuk menyimpan data. Class Box menggunakan tipe data dynamic, sehingga dapat menampung berbagai jenis data. Data yang disimpan awalnya berupa int, kemudian dapat diubah menjadi String. Karena tidak ada pembatasan tipe data, program berpotensi mengalami kesalahan. Kesalahan biasanya terjadi saat program dijalankan (runtime error). Dengan demikian, penggunaan dynamic bersifat fleksibel namun kurang aman.

Kode Program VS Code:


// Nama  : Adril Novra Marcelino
// NIM   : 251410017
// Kelas : SI.2A
    
class Box {
  dynamic data;

  Box(this.data);
}

void main() {
  // Menyimpan data awal
  var box = Box(42);

  // Menampilkan data awal
  int value = box.data;
  print("Nilai awal: $value");

  // Mengubah data
  box.data = "Hello";

  // Menampilkan data setelah diubah
  print("Data setelah diubah: ${box.data}");
}

2.Dengan Generic

  • Pada program ini, Box<T> digunakan sebagai class generic yang dapat menyimpan berbagai tipe data sesuai dengan tipe yang diberikan.
  • Tipe <T> memungkinkan class Box menjadi fleksibel, sehingga dapat digunakan untuk menyimpan data bertipe int, String, atau tipe lainnya.
  • Pada fungsi main(), dibuat dua objek yaitu Box<int> dan Box<String> untuk menyimpan data dengan tipe yang berbeda.
  • Program kemudian menampilkan isi dari masing-masing objek menggunakan print(), sehingga terlihat bahwa setiap Box menyimpan data sesuai dengan tipe yang telah ditentukan.
  • Dengan penggunaan generic, program menjadi lebih aman karena tipe data tidak dapat diubah sembarangan, sehingga menghindari kesalahan tipe data saat program dijalankan.

Kode Program VS Code:


// Nama  : Adril Novra Marcelino
// NIM   : 251410017
// Kelas : SI.2A

class Box {
  T data;

  Box(this.data);
}

void main() {
  // Data awal
  var intBox = Box(42);
  var stringBox = Box("Hello");

  // Menampilkan data
  print("Nilai int: ${intBox.data}");

3.Sintaks Generic Class dan Function

  • Pada program ini, MyClass<T> digunakan sebagai class generic yang dapat menyimpan berbagai tipe data sesuai dengan tipe yang diberikan.
  • Tipe <T> membuat class MyClass menjadi fleksibel, sehingga dapat digunakan untuk menyimpan data bertipe int, String, atau tipe lainnya.
  • Method getData() digunakan untuk mengambil nilai yang tersimpan di dalam class sesuai dengan tipe data yang telah ditentukan.
  • Selain itu, terdapat generic function yaitu myFunc<T> yang dapat menerima dan mengembalikan nilai dengan tipe yang sama, sehingga fungsi ini juga bersifat fleksibel terhadap berbagai tipe data.
  • Pada fungsi main(), dibuat objek MyClass<int> dan MyClass<String>, serta pemanggilan fungsi myFunc dengan tipe int dan String.
  • Program kemudian menampilkan hasil menggunakan print(), sehingga terlihat bahwa setiap data diproses sesuai dengan tipe yang telah ditentukan.
  • Dengan penggunaan generic, program menjadi lebih aman dan terstruktur karena tipe data sudah ditentukan sejak awal, sehingga mengurangi kesalahan saat program dijalankan.

Kode Program VS Code:


// Nama  : ADRIL NOVRA MARCELINO
// NIM   : 251410017
// Kelas : SI2A

class MyClass {
  T data;

  MyClass(this.data);

  T getData() {
    return data;
  }
}

// Generic function
T myFunc(T value) {
  return value;
}

void main() {
  var myInt = MyClass(10);
  var myString = MyClass("test");

  print(myInt.getData());
  print(myString.getData());

  var intValue = myFunc(100);
  var stringValue = myFunc("hello");

  print(intValue);
  print(stringValue);
}

10.Pembatasan Tipe Dengan Class Abstract

  • Class Dog dan Cat mengimplementasikan (implements) class Animal, sehingga wajib mengoverride method makeSound() sesuai dengan perilaku masing-masing hewan.

Kode Program VS Code:


// Nama  : ADRIL NOVRA MARCELINO
// NIM   : 251410017
// Kelas : SI2A

abstract class Animal {
  void makeSound();
}

class Dog implements Animal {
  @override
  void makeSound() {
    print("Woof!");
  }
}

class Cat implements Animal {
  @override
  void makeSound() {
    print("Meow!");
  }
}

class AnimalCage {
  T animal;

  AnimalCage(this.animal);

  void sound() {
    animal.makeSound();
  }
}

void main() {
  var dogCage = AnimalCage(Dog());
  dogCage.sound();

  var catCage = AnimalCage(Cat());
  catCage.sound();

  // var wrong = AnimalCage("test"); // ❌ Error
}

11.Praktik Generic Class & Fungsi

  • Dengan penerapan generic, program menjadi lebih reusable, fleksibel, dan type-safe karena dapat digunakan untuk berbagai jenis data tanpa mengubah struktur kode.

// Repository Generic
class Repository {
  List _items = [];

  void save(T item) {
    _items.add(item);
  }

  T getById(int id) {
    return _items[id];
  }

  List getAll() {
    return _items;
  }
}

// Fungsi swap
void swap(List list, int i, int j) {
  var temp = list[i];
  list[i] = list[j];
  list[j] = temp;
}

// Class Triple
class Triple {
  A first;
  B second;
  C third;

  Triple(this.first, this.second, this.third);
}

void main() {
  var repo = Repository();
  repo.save(10);
  repo.save(20);

  print(repo.getAll());

  var list = [1, 2, 3];
  swap(list, 0, 2);
  print(list);

  var triple = Triple("Age", 21, 1.75);
  print("${triple.first}, ${triple.second}, ${triple.third}");
}

13.Latihan

  • Class Response<T> merupakan generic class yang digunakan untuk menampung hasil proses, baik berupa data maupun error.

Kode Program VS Code:


// Nama  : ADRIL NOVRA MARCELIN0
// NIM   : 251410017
// Kelas : SI.2A

class Stack {
  final List _items = [];

  void push(T value) {
    _items.add(value);
  }

  T? pop() {
    if (_items.isEmpty) return null;
    return _items.removeLast();
  }

  T? peek() {
    if (_items.isEmpty) return null;
    return _items.last;
  }
}

List mergeLists(List a, List b) {
  return [...a, ...b];
}

class Response {
  T? data;
  String? error;

  Response({this.data, this.error});
}

// =======================
// MAIN
// =======================
void main() {
  // Stack
  var stack = Stack();
  stack.push(10);
  stack.push(20);
  print('Peek: ${stack.peek()}');
  print('Pop: ${stack.pop()}');

  // Merge Lists
  var result = mergeLists([1, 2], [3, 4]);
  print('Merge: $result');

  // Response
  var res = Response(data: "Berhasil");
  print('Response: ${res.data}');
}

  • Pada fungsi main(), dibuat dua repository yaitu Repository<String> untuk data pengguna dan Repository<Product> untuk data produk.

Kode Program VS Code:


// Nama  : ADRIL NOVRA MARCELINO
// NIM   : 251410017
// Kelas : SI.2A

// =======================
// GENERIC REPOSITORY
// =======================
class Repository {
  final List _data = [];

  // tambah data
  void add(T item) {
    _data.add(item);
  }

  // ambil semua data
  List getAll() {
    return _data;
  }

  // hapus data
  void remove(T item) {
    _data.remove(item);
  }
}

// =======================
// CONTOH CLASS
// =======================
class Product {
  String name;
  int price;

  Product(this.name, this.price);

  @override
  String toString() {
    return 'Product(name: $name, price: $price)';
  }
}

// =======================
// MAIN
// =======================
void main() {
  // Repository untuk String
  var userRepo = Repository();
  userRepo.add("Alice");
  userRepo.add("Bob");

  print("User Repo: ${userRepo.getAll()}");

  // Repository untuk Product
  var productRepo = Repository();
  productRepo.add(Product("Laptop", 1000));
  productRepo.add(Product("Mouse", 200));

  print("Product Repo: ${productRepo.getAll()}");

  // Hapus data
  userRepo.remove("Alice");
  print("User Repo setelah hapus: ${userRepo.getAll()}");
}
